## System Requirements

- Node.JS
- pnpm

## How to Build

```shell
# Install dependencies.
pnpm i

# Run examples.
pnpx turbo run dev
```

## Monorepo Structure

- `example`: An example project demonstrating how to use `@prisma-korea/prisma-generator-proto` package.
- `packages`
  - `prisma-generator-proto`: Generator logic.
